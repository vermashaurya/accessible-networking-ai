# Accessible Networking AI

A modern, interactive platform showcasing advanced AI logic and networking algorithms. 

## 🚀 Live Demo
This project is configured to be automatically published on GitHub Pages. You can view the live interactive web UI once it is deployed.

## 📂 Project Structure

- `Frontend/UI/`: Contains the complete web interface (HTML, CSS, JavaScript).
- `*.py`: The root directory contains various AI and logic scripts.

## ⚙️ Deployment

A GitHub Actions workflow is included to automatically deploy the contents of the `Frontend/UI` directory to GitHub Pages whenever changes are pushed to the `main` branch.
